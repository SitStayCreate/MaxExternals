chordophone depends on the maxexternals.jar. 
Leave them in the same folder or...

If you are going to use a lot of my max patches,
you may want to copy this to your java-class/lib folder.

The path for this in Max 7 on macOS is: 
/Applications/Contents/Resources/C74/packages/max-mxj/java-classes/lib

You can reach this by right clicking the Max icon via the GUI
